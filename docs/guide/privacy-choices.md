# Privacy Choices

This page describes how to submit privacy requests to Nicasa and what to expect.

## What this is for

Use this page to request access to personal data, request deletion, request portability, or to opt out of certain data uses (for example, "Do Not Sell" requests where applicable).

Note: Nicasa currently does not collect or store personal data (see the Privacy Policy). If you believe we have personal data about you, submit a request and we will investigate.

## How to submit a request

Provide the following information when making a request to help us process it efficiently:

- Your name
- Contact email
- Description of the request (access, deletion, portability, or opt-out)
- Any identifiers that help us locate records (e.g. support ticket ID)

Send requests to one of the following:

- Email: nicasa@w3cub.com
- Open a privacy issue: https://github.com/nicasa-app/nicasa-support/issues

## Response time

We aim to acknowledge and respond to valid requests within 30 days. If we need more time, we will notify you with an estimated timeframe.

## If we do not hold personal data

Because Nicasa is designed to process images locally and not collect user data, many requests will be closed with a confirmation that no personal data is held. If you provide evidence that personal data was collected, we will investigate and take appropriate action.

## Do Not Sell / Opt-out

If you are exercising a jurisdictional right (for example under CCPA/CPRA) to opt out of sale, and you believe we have sold your personal data, please describe the circumstances in your request. If we do not sell personal data, we will confirm that in our response.

## Contact

If you have questions about this process, contact nicasa@w3cub.com or open an issue at https://github.com/nicasa-app/nicasa-support/issues.
